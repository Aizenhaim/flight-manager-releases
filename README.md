# Flight Manager — релизы

Публичные сборки приложения **Flight Manager** (офлайн-бланк рейса бортпроводника).

- `version.json` — последняя версия (versionCode, versionName, ссылка на APK, что нового). Приложение читает его для авто-обновления.
- APK каждой версии — во вкладке **Releases**.

Исходный код — в приватном репозитории. Публикация сюда автоматическая (GitHub Actions при теге `vX.Y.Z`).
