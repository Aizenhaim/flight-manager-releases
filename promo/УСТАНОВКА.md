# Flight Manager — как установить

**Flight Manager** — приложение-бланк рейса для бортпроводников. Бесплатно, работает без интернета.

## Установка (Android)

1. Наведи камеру телефона на **QR-код** (файл `flight-manager-qr.png`) или открой ссылку:
   **https://github.com/Aizenhaim/flight-manager-releases/releases/latest**
2. Скачай файл **FlightManager-….apk** и открой его (тап по скачанному файлу).
3. Android спросит разрешение **«установка из этого источника»** — разреши для браузера или «Файлов», вернись назад и нажми **«Установить»**.

Готово. Значок появится на экране.

## Что внутри
- Заполнение бланка рейса, штампы фактического времени **одним тапом**
- Кнопка **«Обратный рейс»** — новый рейс за секунду
- **Чек-листы** типов ВС (встроенные + свои PDF)
- **История** рейсов и **налёт** с прогрессом к норме
- Всё **офлайн**, без регистрации, данные остаются в телефоне

## Обновления
Внутри приложения: **Настройки → «Проверить обновления»**. Новая версия скачается и установится сама.

## Для iPhone
Пока только Android.
